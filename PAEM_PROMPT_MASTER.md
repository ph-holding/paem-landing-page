# PAEM Real Estate — Website Redesign Context

## Project Context

This document extends the base **Cinematic Landing Page Builder prompt** with specific context for redesigning the official website of **PAEM Real Estate**.

The goal is to redesign the existing website with a **minimalist, brutalist-inspired design system**, prioritizing:

- Large editorial imagery
- Clear information hierarchy
- Very few UI components
- Maximum visual clarity
- Architectural layouts
- Brutalist minimalism

The website should feel **architectural, calm, premium and intentional**, not like a generic corporate real estate site.

---

# Brand

**PAEM Real Estate**

Company focused on:

- Real estate development
- Property rehabilitation
- Residential rentals
- Property sales

The company currently operates in **Zaragoza (Spain)** and develops residential projects such as **La Muela**.

---

# Design Direction

The design should combine:

Minimalism  
Architectural editorial layouts  
Brutalist spacing  
Large typography  
Very large imagery

The site must avoid:

- template-like layouts
- excessive UI components
- card grids everywhere
- marketing-style SaaS design patterns

Instead, prioritize:

- whitespace
- asymmetry
- typography
- large image blocks
- grid layouts
- architectural rhythm

Think of the aesthetic as a mix of:

- architectural portfolio
- minimal editorial magazine
- contemporary brutalist website

---

# Global Website Structure

The new site structure should be **much simpler than the original website**.

Final structure:

There will be **NO separate Contact page anymore**.

Contact information will appear **only at the end of the Home page**.
